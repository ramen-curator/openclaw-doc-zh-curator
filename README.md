# openclaw-doc-zh-curator

自用的OpenClaw的中文翻译文档。

## 背景

问了Gemini 3 Pro ，他表示没发现 OpenClaw 中文翻译有开源贡献的入口 ， 建议我到 Discord 上去问。 而且我又在第一句就对译文不太喜欢，故开此自用项目。  
由于自用缘故，显然不保证正确性、完整性；不保证维护；随时准备归档。  
欢迎 Fork ，在我工作的基础上自行拓展。  
也欢迎以此自建（做了喊我声，不错的话，我就可以不自己做了）。

## 怎么译、为什么这么译

Gemini 3 pro 机翻。
之前翻译过Misskey文档（博文：[《简单讲讲Misskey翻译经历、心得》](https://www.blogger.com/blog/posts/2105424111256398170) ）的经验告诉我，用Gemini 3 pro译已经很够了。
