# openclaw-doc-zh-curator

自用的OpenClaw的中文翻译文档。

## 背景

问了Gemini 3 Pro ，他表示没发现 OpenClaw 中文翻译有开源贡献的入口 ， 建议我到 Discord 上去问。 而且我又在第一句就对译文不太满意，故开此自用项目。  
因为是自用项目，所以不会及时更新、甚至不会更新、修复翻译错误的那种。
欢迎 Fork ，在我工作的基础上自行拓展。
